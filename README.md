# DisasterNet
A Java-based disaster management platform simulating real-world emergency operations centres. Tracks disasters · dispatches rescue teams · manages relief resources · persists full incident logs.

## Java OOP Concepts used
- Classes & objects
- Working with classes and constructors
- Access modifiers, static, final
- Inheritance
- Polymorphism (Overriding and Overloading)
- Nested Classes
- Abstract classes + abstract functions
- Multiple inheritance (interfaces)
- Generic functions + generic classes
- File I/O + streams
- GUI Java Swing [optional]
